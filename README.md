# REST-assignment
 
## เครื่องมือที่ใช้
1. Node.js 
2. Express.js
3. AceBase (nonSQL filebased database)
4. uuid
5. dotenv
6. nodemon

Note: PORT ตั้งค่าที่ไฟล์ .env

## วิธีการติดตั้ง
ให้เปิด Terminal โดย cd ไปที่ directory ของโปรเจค จากนั้นใช้คำสั่ง
```
npm install
```

## วิธี run
ใช้คำสั่ง
```
npm start
```
or
```
npm run start
```
